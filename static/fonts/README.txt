How to use your font
====================

1. Copy the exo_2_regular.css file and fonts/ folder to your website

2. Include the css file using the following link tag:

     <link rel="stylesheet" type="text/css" href="exo_2_regular.css">

3. Reference the font in your CSS using font-family:exo_2_regular

     Example:

     .mytext
     {
        font-family: exo_2_regular;
        font-size:18px;
        color:#aaa;
     }

Important: Please read LEGAL.txt before using this font.
